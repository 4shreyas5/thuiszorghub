# UI Components

This directory contains shadcn/ui components.

## Installation

These components should be generated using shadcn/ui CLI when ready:

```bash
npx shadcn-ui@latest add button
npx shadcn-ui@latest add card
npx shadcn-ui@latest add input
npx shadcn-ui@latest add form
npx shadcn-ui@latest add table
npx shadcn-ui@latest add dialog
npx shadcn-ui@latest add dropdown-menu
npx shadcn-ui@latest add badge
npx shadcn-ui@latest add skeleton
npx shadcn-ui@latest add sheet
npx shadcn-ui@latest add sonner
```

## Components

- **Button** - Primary action component
- **Card** - Container for grouped content
- **Input** - Text input field
- **Form** - Form management with React Hook Form integration
- **Table** - Data table component
- **Dialog** - Modal dialog component
- **Dropdown Menu** - Menu with dropdown options
- **Badge** - Label/badge component
- **Skeleton** - Loading placeholder
- **Sheet** - Side panel component
- **Sonner** - Toast notification system

All components follow shadcn/ui conventions and are built on Radix UI primitives.
